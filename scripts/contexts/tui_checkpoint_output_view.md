# TUI Explore Context: Output View

Goal: validate output viewing for completed parser jobs.

Expected UI state:
- Completed jobs show output artifacts (per-file).
- Errors are visible with file paths and messages.
- Output previews are readable and tied to the correct files.

Explore prompts:
- Open a completed job and view outputs for a few files.
- Verify output file paths or preview text.
- Check error details for failed files.

Report issues:
- Output not visible or mismatched.
- Missing error messages.
- Broken links or empty previews.
